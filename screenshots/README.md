# Screenshots Folder

This folder should contain the following images for the README:

1. **landing-page.png** - Screenshot of the modern landing page with hero section
2. **features.png** - Screenshot showing the role-based dashboards and features
3. **wallet-connect.png** - Screenshot of MetaMask wallet connection interface

## How to Add Screenshots

1. Run your application: `npm start`
2. Open the app in your browser at `http://localhost:3001`
3. Take screenshots using:
   - Windows: `Windows + Shift + S` or Snipping Tool
   - Mac: `Command + Shift + 4`
4. Save the images in this folder with the exact names listed above
5. Commit and push to GitHub:
   ```bash
   git add screenshots/
   git commit -m "Add project screenshots"
   git push
   ```

The images will automatically appear in the README on GitHub!
